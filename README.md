# ScreenOCR

### Install

```
python -m venv .venv
.venv\scripts\activate
pip install -r requirements.txt
```

Then just run

```
python test.py
```

And press `Ctrl + Shift + O` to activate it

Then you drag on your screen, same as a screenshot, and then release and a window with the text will pop up that you can edit or copy or save.

> I made this because I wanted an easy way to copy text from a youtube video when I'm watching courses and such.
>
> Of course it works for anything, not just youtube
